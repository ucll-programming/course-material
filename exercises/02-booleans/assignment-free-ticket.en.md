# Free Ticket

:::TASK
A bus ticket is free for anyone under 12 or over 65.

Write a function `free_ticket(age)` that returns `True` if a person of the given `age` gets a free ticket, `False` otherwise.
:::

:::USAGE

```python
>>> free_ticket(5)
True

>>> free_ticket(6)
False

>>> free_ticket(18)
False

>>> free_ticket(64)
False


>>> free_ticket(65)
True

>>> free_ticket(80)
True
```
