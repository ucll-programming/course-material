# Count Lines

:::TASK
Write a function `count_lines_in_file(path)` that returns the number of lines in the file with the given path.
:::

:::USAGE

```python
>>> source = 'example.txt'
>>> with open(source, 'w') as f:
>>>    f.write('Line 1\nLine 2\nLine 3\n')

>>> count_lines_in_file(source)
3
```
